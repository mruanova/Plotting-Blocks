This website requests all projects to a Web API using Backbone.js and Underscore.js in the front-end and renders the response in a list with markers in Google Maps.

In the back-end a AWS Getway API executes a serverless Lambda function in Node.js that retrieves data from DynamoDB.

Deployment in AWS S3 and Route 53.
